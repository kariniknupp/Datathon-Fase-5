# Datathon-Fase-5
Classificação de Reclamações Financeiras e Análise das Dores dos Clientes
Nesse Tech Challenge nosso desafio foi construir um modelo de Deep Learning capaz de classificar automaticamente o sentimento das reclamações de clientes (positivas ou negativas) com base no texto enviado, e realizar uma análise das principais dores relatadas em cada categoria de produto financeiro.

Analisamos um conjunto de dados com reclamações de consumidores sobre produtos e serviços financeiros, contendo o texto da reclamação, a empresa envolvida e a categoria do produto/serviço (como cartão de crédito, hipoteca, empréstimo, etc.). Fizemos uma análise para ajudar a entender melhor como os consumidores se sentem em relação aos serviços e quais problemas são mais recorrentes.
Verificamos as dores e traçamos estratégias para melhorar a reputação da empresa.
